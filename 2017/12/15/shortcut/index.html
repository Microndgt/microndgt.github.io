快捷键大全 - Mac

2017-12-15 23:57:57
  • 编程

仅列出自己觉得有用并且常用的快捷键

VSCode

基本控制

  • shift + command + p 打开命令面板
  • command + p 快速打开
  • shift + command + n 新建窗口
  • command + w 关闭文件窗口
  • shift + command + w 关闭窗口
  • shift + command + c 打开外部shell
  • ctrl + r 切换工作区(项目)
  • command + k + commad + s 打开键盘快捷方式设置

编辑

  • alt + 上/下 移动一行
  • shift + alt + 上/下 复制一行
  • shift + command + k 删除当前行
  • command + enter 下一行插入
  • shift + command + enter 上一行插入
  • command + [/] 缩进，反缩进
  • command + 上/下 文档首，尾
  • command + 左/右 行首，尾
  • shift + command + 左/右 选择光标左边或者右边的文本
  • shift + command + 上/下 选择光标上面或者下面的文本
  • shift + alt + 左/右 选择光标左边或者右边的单词
  • command + / 行注释
  • shift + alt + a 块注释
  • command + I 选择当前行
  • ctrl + d 向后删除
  • ctrl + a 行首
  • ctrl + e 行尾
  • command + k + command + 0 全部折叠
  • command + k + command + j 全部展开
  • command + k + command + ] 递归的局部展开
  • command + k + command + [ 递归的局部折叠
  • control + k 删除右面所有内容
  • command + backspace 删除左面所有内容
  • command + d 同样的单词多选，一个个选中
  • shift + command + l 所有的匹配项都选中

多选

  • alt + click 插入光标，有选择性的多选光标
  • alt + command + 上/下 在上面或者下面增加光标

搜索替换

  • alt + command + f 替换
  • command + g 下一个
  • shift + command + g 上一个
  • alt + enter 选择所有查找项

导航

  • control + g 跳到第几行
  • control + - 后退
  • shift + control + - 前进

编辑器

  • command + \ 切分窗口
  • ctrl + tab 文件窗口切换
  • shift + command + [/] 切换文件选项卡
  • alt + command + 左/右 切换文件窗口
  • command + 2/1/3 开始或者切换第二个(同样)文件窗口

显示

  • control + command + F 全屏
  • command + b 显示或者不显示侧边栏
  • shift + command + e 显示资源管理器
  • command + enter 在文件上敲击这个，会打开文件，直接enter修改名字
  • shift + command + x 显示扩展
  • shift + command + d 显示调试
  • ctrl + shift + g 显示git
  • shift + command + U 显示输出 再次按就消失
  • shift + command + f 显示搜索
  • control + · 打开集成终端 再次按就消失

VSCode的Python配置

配合conda来进行环境的管理，比如一个项目cat, 可以conda create -n cat python=3.6创建虚拟环境，然后shift+command+p，输入python: select interpreter来选择刚才配置的虚拟环境，这样就会在项目下面生成一个.vscode，不过这个需要加入.gitignore。

这样f5就可以开始调试了, shift+command+d开启调试页面，可以看到调试变量。另外在终端也可以键入source activate cat启动虚拟环境，进行pip等管理或者运行。
